== Installation ==

1. Upload the `wp-twitter-autopost` plugin to your `/wp-content/plugins/` directory.
2. Activate the plugin using the `Plugins` menu in WordPress.
3. Go to Settings > WP Twitter Autopost.
4. Adjust the WP Twitter Autopost Options as per your needs. 
5. Create a Twitter application from twitter developrs website and Configure your OAuth setup keys.
